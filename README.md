# erc20_contract

- Run contract locally:
  
```bash
npx hardhat node
```

- Deploy locally:

```bash
npx hardhat run --network localhost scripts/deploy.js
```

- Run interact script

```bash
npx hardhat run --network localhost scripts/interact.js
```
